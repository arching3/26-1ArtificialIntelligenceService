# 아주 쉬운 인수인계 요약

이 프로젝트는 회사 공시를 모아서, 사람이 질문하면 AI가 그 공시를 보고 대답해 주는 프로그램입니다.

## 지금 만든 것

- 새 버전 프로그램은 `src/` 폴더에 있습니다.
- 화면은 `streamlit_app.py`로 켭니다.
- 예전 실험용 코드는 `legacy/` 폴더에 따로 넣어 두었습니다.
- 삼성전자 공시로 실제 테스트도 해 봤고, 질문 답변까지 성공했습니다.

## 켜는 방법

먼저 비밀 열쇠 2개가 필요합니다.

- `DART_API_KEY`
- `OPENAI_API_KEY`

그다음 백엔드와 화면을 각각 켭니다.

```bash
source ../venv.sh
uvicorn src.api_server:app --host 127.0.0.1 --port 8000 --reload
streamlit run streamlit_app.py
```

## 하는 일

- 회사 이름을 찾습니다.
- DART에서 회사 공시를 가져옵니다.
- 공시 내용을 잘게 나눠 저장합니다.
- 질문과 관련 있는 내용을 찾습니다.
- AI가 그 내용을 보고 답합니다.
- 주식 사라, 팔아라, 목표가는 얼마다 같은 투자 조언은 막습니다.

## 아직 남은 일

- 화면을 더 직접 확인하고 다듬기
- 관심 기업 목록을 진짜 DB에 저장하기
- 로그인 기능 제대로 붙이기
- 가짜 주가 데이터를 실제 주가 데이터로 바꾸기
- 긴 작업 상태를 더 안전하게 관리하기
- 에러 메시지를 더 쉽게 보여 주기
- 평가용 데이터와 테스트 흐름 정리하기
