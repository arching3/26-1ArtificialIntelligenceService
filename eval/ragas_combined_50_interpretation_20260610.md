# RAGAS 50문항 통합 평가 해석

## 평가 구성

- 수정 질문 목록: 10문항
- 인덱스 원문 기반 신규 질문: 40문항
- 합계: 50문항
- 평가 지표: Faithfulness, Answer Relevancy, Context Precision, Context Recall

## 통합 점수

| 지표 | 점수 |
|---|---:|
| Faithfulness | 0.9027 |
| Answer Relevancy | 0.5488 |
| Context Precision | 0.6491 |
| Context Recall | 0.8600 |

## 데이터셋별 점수

| 데이터셋 | 문항 수 | Faithfulness | Answer Relevancy | Context Precision | Context Recall |
|---|---:|---:|---:|---:|---:|
| 수정 질문 목록 | 10 | 0.9833 | 0.6144 | 0.7247 | 0.8000 |
| 인덱스 원문 기반 | 40 | 0.8826 | 0.5323 | 0.6302 | 0.8750 |

## 해석

### Faithfulness

0.9027로 높다. 생성된 답변 대부분이 검색된 공시 Context에 의해 뒷받침되며, Context에 없는 내용을 임의로 추가하는 현상은 제한적이다.

다만 일부 답변은 원문에 없는 제품 기능 설명을 덧붙이거나, 질문이 요구한 기간과 다른 수치를 제시했다. 따라서 높은 평균이 모든 개별 답변의 사실 정확성을 보장하지는 않는다.

### Answer Relevancy

0.5488로 네 지표 중 가장 낮다. 답변이 공시 내용에 근거하더라도 질문이 요구한 대상, 범주 또는 기간에 정확히 대응하지 못하는 사례가 존재한다.

대표적으로 승용 차종 질문에 RV 차종을 제시하거나, 특정 반기·분기의 수치 대신 누적 수치를 답하는 문제가 확인됐다.

### Context Precision

0.6491로 보통 수준이다. 필요한 근거가 검색 결과에 포함되는 경우가 많지만, 상위 검색 Context에 질문과 직접 관련 없는 청크도 함께 포함된다.

검색 결과의 불필요한 청크는 LLM이 잘못된 차종, 기간 또는 범주를 선택할 가능성을 높인다. 질문 의도와 메타데이터를 활용한 재정렬이 필요한 상태다.

### Context Recall

0.8600으로 높다. 정답 작성에 필요한 근거는 대체로 검색 Context 안에 포함된다.

다만 기아의 승용·RV·상용 차종과 과거 분기 재무 질문에서 필요한 근거가 누락되거나 다른 기간의 자료가 선택되면서 점수가 낮아졌다.

## 종합 판단

현재 시스템은 공시 근거를 찾아 답변을 생성하는 기본 능력은 양호하다. 특히 근거 충실도와 검색 재현율은 높은 편이다.

주요 개선 대상은 다음 두 가지다.

1. 검색된 여러 Context 중 질문과 직접 관련된 청크를 상위에 배치하는 정밀도
2. 질문의 제품 범주와 보고 기간을 정확히 구분하여 답변하는 관련성

따라서 현재 결과는 "근거는 대체로 찾지만, 가장 적합한 근거를 골라 질문에 정확히 답하는 단계에서 손실이 발생한다"고 해석할 수 있다.

## 주의사항

- 수정 질문 10문항과 신규 40문항은 문항 구성과 난이도가 다르므로 단순 비교에는 한계가 있다.
- 통합 점수는 50문항 전체의 문항 수 가중 평균이다.
- 평가 대상 기업과 인덱스 상태가 제한되어 있어 전체 기업 성능으로 일반화할 수 없다.
- RAGAS는 LLM Judge와 임베딩을 사용하므로 재실행 시 점수가 일부 달라질 수 있다.
